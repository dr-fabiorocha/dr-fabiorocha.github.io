This research proposes an extension on the Raspberry Pi Weather Station so the station is now able to possess more sensors than originally planned. The extension method uses wireless sensor modules to send data samples using the 802.11 protocol and also modifies the station’s software setup to run as an access point.

“EXTENDING THE RASPBERRY’S WEATHER STATION BY USING WIRELESS SENSORS”, Fábio Rodrigues de la Rocha, Gabriel Carvalho, International Journal of Research in Computer & Information Technology (IJRCIT), Vol. 4, Issue 2, March-2019
